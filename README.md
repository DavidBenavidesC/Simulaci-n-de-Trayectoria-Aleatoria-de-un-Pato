# Simulación de Trayectoria Aleatoria de un Pato

Simulación sencilla desarrollada en NetLogo en la que un pato amarillo inicia en una zona roja y se desplaza aleatoriamente por el entorno, dejando un rastro a su paso. La simulación finaliza cuando el pato alcanza la zona azul, la cual cambia a color verde.

## Características

* Crea un mundo de 50 × 50.
* Define una zona de inicio roja y una zona objetivo azul.
* Genera un pato amarillo en la zona de inicio.
* Desplaza al pato de forma aleatoria.
* Dibuja un rastro amarillo que representa el recorrido realizado.
* Cambia la zona objetivo de azul a verde cuando es alcanzada.
* Detiene la simulación automáticamente al llegar al destino.

## Archivos

* `text.txt`: contiene el código completo del modelo en NetLogo.

## Requisitos

* NetLogo

## Cómo ejecutar

1. Abrir el archivo `.nlogo` en NetLogo.
2. Ejecutar `setup` para inicializar el entorno.
3. Ejecutar `go` para iniciar la simulación.
4. La simulación terminará automáticamente cuando el pato llegue a la zona objetivo.
